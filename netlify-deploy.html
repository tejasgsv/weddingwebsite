🚀 Deploy Your Wedding Website in 2 Minutes

Step 1: Go to Netlify

Open this link in a new tab:

Open Netlify Drop

Step 2: Drag & Drop

1. Select ALL files in your wedding folder (Ctrl+A)

2. Drag them to the Netlify Drop zone

3. Wait for upload to complete

Step 3: Get Your Link

Netlify will give you a URL like:

https://amazing-wedding-123.netlify.app

Your website is now LIVE! 🎉

Alternative: GitHub Pages

1. Create GitHub repository: wedding-website

2. Upload all files

3. Enable Pages in Settings

4. Get URL: https://yourusername.github.io/wedding-website
